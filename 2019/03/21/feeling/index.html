小感慨而已

Posted on 2019-03-21 |
| Words count in article 1.3k
告别的时候，总会有许多感慨～～～

最后一两天了，坐在工位上，就随便发点小感慨吧～～

序

人总是喜欢回忆的，该要离开的时候，去审视周围的一切，会愈发的去珍惜。这段时间，不断的在想，如果存在平行时空，那么在人生的一些关键节点，做了不同选择的我在另外一个平行时空，究竟过着怎么样的生活呢？比如，当初选择了国企，是不是现在已经丧失了还要奋斗的心？再比如，没有读研，是不是就已经结婚生子，过着柴米油盐的日子了？

记得我问过高中班主任，有些事情明明可以不去做，没有人会指责的，为什么还那么拼命？他说：因为不甘心，不甘心这样屈于平凡，不甘心现在的这番成就…想必，能走到现在的我，也是因为不甘心吧。不甘心，所以选择了进入发展最快的互联网行业，不甘心，所以还想再拼一把～

关于北京

北京是一个公平的城市，付出了多少，便会收获多少。去拼命、去努力，就会看到结果，虽然有时结果不那么尽如人意。只是这也是一个容易让人不敢有丝毫松懈的城市，看着周围的人都在拼命，你如果不去努力，不去学习，就会被淘汰。

来这里的第一年，就见证了各大共享单车、乐视的倒闭，见证了百度外卖、饿了么、摩拜的被收购。紧接着的一年，是知乎、美团、网易、滴滴、百度等各大互联网公司的裁员，阿里系、字节跳动的扩张，未来的十年，是否是大数据、区块链、AI的十年，尚未可知～～所以，它让人丝毫不敢停下前进的脚步。这样的一座城市，对于没有任何背景、任何依靠的我而言，无疑是最好的选择，我可以在这里去努力实现自己的梦想，哪怕苦了累了，也值了～

关于讯飞乐知行

在职期间，会有无数的吐槽，感觉想要吐槽的人和事，三天三夜都讲不完。该走的时候，又感觉其实这些又都有什么关系呢？哪个公司的体系没有自己固有的毛病，哪个公司没有自己看不惯的一些事和人，所以，也都无所谓了～～换工作，无非是从一个坑跳进另外一个坑，平常心对待一切，就好了～

或许是自己的第一份工作，职场经验并不足，外加公司的研发氛围很和谐，人际关系简单，给自己造成了一种错觉，过于松懈，没有花任何心思在这上边。那天，领导的一句话突然点醒了我，或许你的工作量、工作能力很突出了，但是为什么得到的待遇和别人不一样呢？因为在面对较大工作量的时候，你和别人的态度差别太大。领导需要看到的是，对于有挑战性的工作，你欣然接受，并及时汇报情况，虽然可能不能严格按照研发计划完成，但是你任劳任怨的加班了、努力去完成了，领导看到你的付出了，就会认可你的工作量。而你是怎么做的呢？抱怨+加班，最后也保质保量的完成了，领导会有什么感受呢？所以，工作中，少抱怨，多干活，多汇报～～

面试中，有不少面试官问，怎么评价你的上一家公司？嗯，感觉乐知行更像一个学校，离职是因为我该从这个学校毕业了。从对前端有兴趣，进入乐知行，到想更加深入的去了解前端，继续从事该行业，并在该行业中有所建树，感觉这就是他对我而言的意义吧～～

关于人和事

大家都说我18年的下半年进步好大，承担着好几个项目。可是只有自己才明白，下半年自己的学习已经很缓慢了，感觉能从项目中得到的知识点越来越少。但是这对于一个从事前端工作才刚刚一年的我而言，是个多么可怕的事情。或许是之前的积累，才有了后边的爆发，渐渐的，感觉自己能摸索到如何去独立解决问题。感恩，选科见证了我在前端方面的成长～～

对于下份工作的期许

感觉有位面试官说的也有一定的道理，如果你想在前端领域游刃有余的话，需要在前端纵深的同时，横向扩展学习能力也需要具备。比如：react + 小程序，甚至于 node.js 和 python。所以，期待我的下份工作带我不一样的全新视野。也希望自己可以有机会在项目中实践一下 react 的全家桶～～

嗯嗯，就感慨到这里吧，感恩所拥有的一切～～
